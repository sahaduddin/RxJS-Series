🔹 What is a Promise in JavaScript?

A Promise in JavaScript is an object that represents the eventual completion (or failure) of an asynchronous operation and its resulting value.

👉 Simply: A promise is like a guarantee of a future value that you don’t have right now but will receive later.

It has three states:

  1. Pending → The operation is still running.
  2. Fulfilled (Resolved) → The operation finished successfully, and you got the result.
  3. Rejected → The operation failed, and you got an error.

3. User Authentication

async function loginUser(username, password) {{ '{' }}
  try {{ '{' }}
    const response = await fetch('/api/login', {{ '{' }}
      method: 'POST',
      body: JSON.stringify({{ '{' }} username, password {{ '}' }})
    {{ '}' }});
    
    if (!response.ok) {{ '{' }}
      throw new Error('Login failed');
    {{ '}' }}
    
    const user = await response.json();
    return user;
  {{ '}' }} catch (error) {{ '{' }}
    console.error('Login error:', error);
    throw error;
  {{ '}' }}
{{ '}' }}

Used in login systems, OAuth flows, and secure API authentication
